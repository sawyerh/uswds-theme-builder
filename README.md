## Repo structure

```
├── server    ⚡️ Sass compilation server

```

## Project setup

**Prerequisites:**

- [Node.js](https://nodejs.org/en/)
- [Dart](https://dart.dev/)

**Install dependencies:**

```
npm run setup
```

## Local development

```
npm run dev
```
