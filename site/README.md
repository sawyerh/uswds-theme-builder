## Local development

Prerequisites:

- Node v14+
- NPM v7+

**Install dependencies:**

```sh
npm install
```

**Run the site:**

The site depends on a Sass compilation server running locally as well, on port `8080`.

```sh
npm run dev
```

## Deploy

```
npm run deploy
```
